# Project File Structure — SafeExam

```
safe-exam/
├── client/        # Frontend (React / Next.js)
├── server/        # Backend (Node.js / APIs)
├── docs/          # Documentation
├── database/      # Schema and queries
├── .env.example
├── README.md
```

---

## 📌 Rules

* Keep frontend and backend separate
* Store reusable components properly
* Maintain clean folder hierarchy
* Do not mix business logic with UI
