# Exam Flow — SafeExam

---

## 🧪 Steps

1. Student logs in
2. Enters exam code / URL
3. System validates exam
4. Instructions screen
5. Exam starts
6. Timer begins
7. Answers saved automatically
8. Student submits exam
9. Result generated

---

## ⏱ Features

* Auto-submit on timeout
* Navigation between questions
* Secure exam environment

---

## 🎯 Objective

Smooth and secure exam experience.
